# Requisitos Não Funcionais (RNF)

## RNF01 - Desempenho
O sistema deve carregar as páginas em até 3 segundos.

## RNF02 - Responsividade
O sistema deve ser responsivo e adaptável a dispositivos móveis e desktops.

## RNF03 - Usabilidade
A interface deve ser intuitiva e de fácil navegação.

## RNF04 - Compatibilidade
O sistema deve funcionar nos principais navegadores modernos (Chrome, Firefox, Edge).

## RNF05 - Manutenibilidade
O código deve ser estruturado e organizado para facilitar manutenção.

## RNF06 - Escalabilidade
O sistema deve permitir expansão futura para funcionalidades como CRUD completo.

## RNF07 - Segurança (Futuro)
O sistema deve garantir proteção básica de dados dos usuários no fórum.

## RNF08 - Disponibilidade
O sistema deve estar disponível continuamente para acesso dos usuários.
