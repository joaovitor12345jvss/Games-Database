# Requisitos Funcionais (RF)

A Tabela apresenta os Requisitos Funcionais elicitados para o sistema Games Database.

| ID | Requisito Funcional | Descrição | Prioridade |
|----|----------------------|-----------|------------|
| RF01 | Visualizar Jogos Populares | O sistema deve exibir uma lista de jogos populares na página inicial. | Alta |
| RF02 | Buscar Jogos | O sistema deve permitir que o usuário pesquise jogos pelo nome através de uma barra de busca. | Alta |
| RF03 | Filtrar por Categoria | O sistema deve permitir filtrar jogos por gênero (ação, aventura, RPG, estratégia, etc.). | Média |
| RF04 | Visualizar Detalhes do Jogo | O sistema deve exibir informações detalhadas de um jogo selecionado. | Alta |
| RF05 | Navegação Responsiva | O sistema deve permitir navegação fluida entre páginas/seções. | Alta |
| RF06 | Fórum de Discussão | O sistema deve permitir que usuários acessem uma área de fórum para discussões. | Alta |
| RF07 | Criar Post no Fórum | O sistema deve permitir que usuários criem postagens no fórum. | Alta |
| RF08 | Visualizar Postagens | O sistema deve permitir visualizar postagens feitas por outros usuários. | Alta |
| RF09 | Sistema CRUD de Jogos (Futuro) | O sistema deve permitir cadastrar, editar, excluir e visualizar jogos. | Baixa |
| RF10 | Interface Interativa | O sistema deve atualizar conteúdos dinamicamente sem recarregar a página (quando possível). | Média |
