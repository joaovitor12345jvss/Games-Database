# Requisitos Não Funcionais (RNF)

A Tabela apresenta os Requisitos Não Funcionais do sistema Games Database.

| ID | Requisito | Descrição | Prioridade |
|:--:|-----------|-----------|:----------:|
| RNF01 | Desempenho | O sistema deve carregar as páginas em até 3 segundos. | Alta |
| RNF02 | Responsividade | O sistema deve ser responsivo e adaptável a dispositivos móveis e desktops. | Alta |
| RNF03 | Usabilidade | A interface deve ser intuitiva e de fácil navegação. | Alta |
| RNF04 | Compatibilidade | O sistema deve funcionar nos principais navegadores modernos (Chrome, Firefox, Edge). | Média |
| RNF05 | Manutenibilidade | O código deve ser estruturado e organizado para facilitar manutenção. | Média |
| RNF06 | Escalabilidade | O sistema deve permitir expansão futura para funcionalidades como CRUD completo. | Média |
| RNF07 | Segurança (Futuro) | O sistema deve garantir proteção básica de dados dos usuários no fórum. | Média |
| RNF08 | Disponibilidade | O sistema deve estar disponível continuamente para acesso dos usuários. | Alta |
